
This directory contains a sample input file (input.txt) and six
example output files.

File           Description
-----------------------------------------------------------------
vsp_first.out  VSP first-fit with memory size = 2000
vsp_best.out   VSP best-fit with memory size = 2000
vsp_worst.out  VSP worst-fit with memory size = 2000
P100.out       Paging with memory size = 2000 and page size = 100
P200.out       Paging with memory size = 2000 and page size = 200
P400.out       Paging with memory size = 2000 and page size = 400
